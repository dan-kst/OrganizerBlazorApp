# Organizer App
A robust, multimedia-supported task management system built with Blazor and .NET 10. This project was designed to handle complex task hierarchies, media attachments, and automated business logic, providing a seamless desktop-like experience on the web.

# Getting Started
## Prerequisites
* .NET SDK (v8.0 or later)
* SQLite

### Installation

Clone the repository:

    git clone https://github.com/dank-st/OrganizerBlazorApp.git

Navigate to the project directory:

    cd OrganizerBlazorApp

Update the database:

    dotnet ef database update

Run the application:

    dotnet run
